# 闲鱼数码硬件行情系统（第一阶段）

## 1. 项目简介

本项目是一个面向数码硬件的**二手行情采集与可视化系统**。
第一阶段仅聚焦 **闲鱼二手市场**，围绕一组**固定硬件清单**，每天定时采集公开可见的商品列表数据，对结果进行清洗、去重、异常值过滤和统计分析，最终在前端页面中可视化展示：

* 今日参考价格
* 近 7 / 30 / 90 天价格走势图
* 当日样本数
* 价格区间
* 代表性在售样本
* 简单行情判断（低位 / 正常 / 偏高）

本项目的核心目标不是做“全站最低价爬虫”，而是做一个**面向固定硬件池的二手行情看板**。

---

## 2. 第一阶段目标

第一阶段只做以下内容：

* 仅采集 **闲鱼**
* 仅维护 **固定硬件池**
* 每天 **定时爬取一次**
* 仅采集 **公开可见的商品列表结果**
* 对数据做基础清洗与聚合
* 展示 **首页行情卡片** 和 **商品详情页走势图**
